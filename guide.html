Before You Begin

WARNING

NetVault-Tactical stores encrypted configuration data that may reveal network topology. Zero-trust authentication (ICAM) must be enabled at every installation, and TPM-backed encryption must never be disabled without authorization.

CAUTION

Restoration overwrites the current running configuration on the target device. Confirm the target is a replacement unit, or that the current configuration can be discarded, before restoring.

Quick Start

  1. Confirm the host server meets the hardware and software requirements on the Application Prototype page.
  2. Install Docker Engine and enable TPM 2.0 in BIOS/UEFI.
  3. Pull the container image and initialize the database and key store.
  4. Start the application stack and access the dashboard at https://<server-ip>:8443.
  5. Register your first network node and configure a backup schedule.

Full step-by-step commands are included in the downloadable User Guide.
